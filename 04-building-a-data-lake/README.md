# Building a Data Lake

## Getting Started


### 1. change directory to project 04-building-a-data-lake
```sh
$ cd 04-building-a-data-lake/
```

### 2. applying code for saving jupyter lab
```sh
$ sudo chmod 777
```

### 3. prepare environment workspace
```sh
$ docker-compose up
```

<br>
